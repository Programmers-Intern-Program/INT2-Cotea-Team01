# 코티 (Cotea)

프로그래머스 코딩테스트 풀이를 돕는 AI 힌트 제공 Chrome 익스텐션입니다.
정답을 바로 알려주는 대신, **키워드 → 구현 순서 → 방향 전환 → 자유 질의** 4단계로 나눠 방향성을 제시합니다. Java 문제만 지원합니다.

## 왜 만들었나

코딩테스트를 준비할 때 막히면 보통 둘 중 하나를 하게 됩니다 — 검색해서 답을 그대로 베끼거나, 너무 오래 붙잡고 있다가 포기하거나. 코티는 그 사이 지점을 겨냥합니다. 문제를 다 풀어주지 않으면서도, 막힌 지점에 맞는 만큼만 힌트를 줍니다.

## 주요 기능

- **4단계 힌트**: 키워드 → 구현 순서 → 방향 전환 → 자유 질의
- **단계별 대응**: 풀기 전 / 풀이 중 막힘 / 오답 진단
- **RAG 기반 힌트 생성**: 문제별 메타데이터 + 일반 알고리즘 지식 베이스를 결합해 답변 생성

## 기술 스택

| 영역        | 기술                                            |
| ----------- | ----------------------------------------------- |
| 백엔드      | Spring Boot                                     |
| 프론트엔드  | Chrome Extension (Manifest V3)                  |
| AI          | Gemini (`gemini-embedding-001`) 임베딩 기반 RAG |
| 벡터 DB     | Chroma (프로토타입)                             |
| 대상 플랫폼 | Programmers (Java)                              |
